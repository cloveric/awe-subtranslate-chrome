# Awe SubTranslate 隐私权政策

生效日期：2026 年 2 月 17 日

Awe SubTranslate（以下简称“本扩展”）用于网页文本与视频字幕翻译。本文说明本扩展处理哪些数据，以及如何使用这些数据。

## 1. 我们处理的数据

本扩展可能处理以下信息：

1. 网页可见文本、用户选中文本、字幕文本
用途：提供翻译结果。
2. 浏览器本地设置数据
示例：目标语言、翻译服务、显示主题、字幕开关、服务配置（如 API Key、模型名、Base URL）。
3. 本地调试日志
仅用于浏览器开发者工具中的本地调试。

## 2. 数据使用方式

1. 仅用于实现用户主动触发的翻译功能。
2. 当用户发起翻译时，相关文本会直接发送到用户选择的第三方翻译服务商。
3. 开发者不会将该数据用于广告、画像或出售用户数据。

## 3. 第三方服务

根据用户选择，翻译请求可能发送至：

- Google Translate
- Microsoft Translator
- DeepL
- OpenAI
- Anthropic Claude
- Google Gemini
- DeepSeek

开发者不通过自建后端服务器中转翻译流量。第三方服务对提交数据的处理受其自身条款与隐私政策约束。

## 4. Chrome 权限与用途

- `storage`：在本地保存用户偏好与服务配置。
- `contextMenus`：提供右键菜单的页面/选中文本翻译入口。
- `activeTab`：在用户交互后对当前活动标签页执行翻译操作。
- `host_permissions (<all_urls>)`：在用户访问的网站中运行内容脚本以实现翻译功能。

## 5. 数据保留与删除

1. 开发者不会在自有服务器保存用户翻译内容或 API Key。
2. 本地设置保存在 `chrome.storage.local`。
3. 用户可通过扩展重置、清理浏览器扩展数据或卸载扩展删除本地数据。
4. 第三方翻译服务的数据保留策略由其自身决定。

## 6. 安全

本扩展与翻译服务通信时使用 HTTPS。请避免提交高度敏感内容到在线翻译服务。

## 7. 未成年人

本扩展不专门面向 13 岁以下儿童，也不会故意收集儿童个人信息。

## 8. 政策更新

本政策可能因产品或合规变化而更新，以最新发布版本及生效日期为准。

## 9. 联系方式

如有隐私问题，请通过以下方式联系：

- GitHub Issues: https://github.com/cloveric/awe-subtranslate-chrome/issues

本扩展的数据使用遵守 Chrome Web Store User Data Policy，并适用其 Limited Use 要求。
